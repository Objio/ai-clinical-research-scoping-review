# Data Directory

This directory contains data files used in the analysis.

## Data Access

Due to size constraints, the raw data files are not included in this repository.

### Available on Request:
- `scopus_export_2023_2025.csv` - Original Scopus export (8,522 documents)
- `processed_corpus.csv` - Cleaned corpus (8,395 documents with valid text)

### To obtain the data:
1. Contact the author at [email]
2. Or replicate the Scopus search using the query in the manuscript
